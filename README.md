# Dynamic Data Table Manager

This project is a web application built using Next.js, React, and Redux Toolkit.
It allows users to manage data in a table format with features like editing, adding, deleting, searching, and exporting data.

Features

Add, edit, and delete rows

Search through table data

Sort columns in ascending or descending order

Manage columns (add new columns or hide/show existing ones)

Inline editing by double-clicking on cells

Import data from a CSV file

Export table data to a CSV file

Pagination for large datasets

Saves column visibility preferences locally

Technologies Used

Next.js

React

TypeScript

Redux Toolkit

Material UI

PapaParse (for CSV import/export)

FileSaver.js

UUID

How to Run

Download or clone the repository

git clone https://github.com/your-username/Dynamic-Data-Table-Manager.git


Open the folder

cd Dynamic-Data-Table-Manager


Install dependencies

npm install


Start the development server

npm run dev


Open the app in your browser

http://localhost:3000


How It Works

The data is stored and managed using Redux Toolkit.

The user can modify data directly in the table.

All visible columns and table data are managed through Redux state.

The app supports importing and exporting CSV files for data handling.

Column visibility is saved using local storage.
